## Технологический стек

Наше приложение построено с использованием следующих технологий:

- **React**: Библиотека JavaScript для создания пользовательских интерфейсов. Позволяет создавать повторно используемые UI-компоненты.
- **TypeScript**: Надмножество JavaScript, которое добавляет статические типы, улучшая качество и поддерживаемость кода.
- **Vite**: Быстрый инструмент сборки, который обслуживает ваш код с использованием нативных импортов ES-модулей во время разработки, обеспечивая быструю и плавную работу.
- **Redux Toolkit**: Набор инструментов и лучших практик для упрощения настройки и использования Redux для управления состоянием.
- **RTK Query**: Мощный инструмент для получения и кэширования данных, являющийся частью Redux Toolkit, упрощая интеграцию API.
- **styled-components**: Библиотека для стилизации компонентов React с использованием шаблонных литералов, позволяющая создавать скопированные и динамические стили.

## Функциональные возможности

- **Аутентификация пользователей**: Безопасный вход и регистрация для пользователей.
- **Профили специалистов**: Просмотр подробных профилей специалистов.
- **Публикация вакансий**: Размещение вакансий и управление откликами.
- **Уведомления в реальном времени**: Оставайтесь в курсе с помощью уведомлений в реальном времени.
- **Адаптивный дизайн**: Полностью адаптивный дизайн, обеспечивающий удобство использования на всех устройствах.

## Установка и настройка

Чтобы начать работу с нашим приложением, выполните следующие шаги:

1. **Клонируйте репозиторий**

    ```bash
    git clone https://github.com/yourusername/your-repo-name.git
    cd your-repo-name
    ```

2. **Установите зависимости**

   Убедитесь, что у вас установлены `node` и `npm`. Затем выполните:

    ```bash
    npm install
    ```

3. **Запустите приложение**

   Запустите сервер разработки:

    ```bash
    npm run dev
    ```

   Приложение будет доступно по адресу `http://localhost:3000`.

## Использование

После настройки приложения вы можете начать использовать его для поиска и найма специалистов:

1. **Регистрация**: Создайте учетную запись или войдите, если у вас уже есть аккаунт.
2. **Поиск специалистов**: Используйте функцию поиска, чтобы найти специалистов, соответствующих вашим критериям.
3. **Публикация вакансий**: Если вы ищете сотрудников, разместите свои вакансии.
4. **Отклик на вакансии**: Специалисты могут откликаться на опубликованные вакансии, а работодатели могут просматривать отклики.
